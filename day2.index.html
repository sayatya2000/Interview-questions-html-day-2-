1.Difference between HTML tags and elements.

2.Explain br tag.


br tag is a block level element. It take the full width of the web page. It's also a self closing tag. All elements will occur after the br tag will be sent to the next line.


3.Explain how many different types of lists we have in html.


There is 3 types of list in HTML.


  1. Ordered List or Numbered List
  2. Unordered List or Bulleted List
  3. Description List or Defination List

4.What are semantic tags?


semantic tags are tags are tags that define the meaning of the content they contain. eg: image tag,button tag, heading tag and anchor tag etc.
